¿Qué hace el sistema?

Funcionalidades del Proyecto

Un sistema completo para evaluar, gestionar y mitigar riesgos en proyectos de tecnología

🎯

Predicción Inteligente de Riesgo

Ingresa las características de tu proyecto y el modelo XGBoost predice automáticamente:

  • Nivel de riesgo general: Alto, Medio o Bajo
  • Probabilidad exacta de sobrecosto
  • Probabilidad exacta de retraso
  • Probabilidades por cada clase de riesgo
📋

Gestión CRUD de Proyectos

Administra los proyectos en ejecución con operaciones completas:

  • Registrar proyectos después de evaluarlos
  • Tabla interactiva con todos los proyectos activos
  • Editar y actualizar datos de proyectos
  • Eliminar proyectos con confirmación
  • Finalizar y mover al dataset de entrenamiento
📄

Reportes PDF Profesionales

Genera reportes automáticos con toda la evaluación del riesgo:

  • Datos completos del proyecto evaluado
  • Resultados con probabilidades detalladas
  • Interpretación personalizada según nivel de riesgo
  • Recomendaciones específicas de mitigación
  • Descarga directa o envío por email
🔄

Reentrenamiento Continuo

El sistema mejora con cada proyecto finalizado:

  • Proyectos finalizados alimentan el dataset
  • Botón de reentrenamiento en la interfaz
  • GridSearchCV optimiza hiperparámetros
  • Modelos se recargan en memoria automáticamente

Flujo de Uso

¿Cómo Funciona?

El proceso completo desde la evaluación hasta la gestión de tu proyecto

1

Ingresa los datos del proyecto

Selecciona el tipo de proyecto, metodología, duración estimada, presupuesto, recursos, tecnologías, complejidad, experiencia del equipo y número de hitos clave.

2

Obtén la predicción de riesgo

El modelo XGBoost analiza las 9 variables y retorna el riesgo general (Alto/Medio/Bajo), las probabilidades de sobrecosto y retraso, y las probabilidades por clase.

3

Genera un reporte PDF

Descarga un reporte profesional con los datos del proyecto, predicción detallada e interpretación con recomendaciones. También puedes enviarlo por email directamente.

4

Guarda y gestiona el proyecto

Registra el proyecto en ejecución para darle seguimiento. Edita, elimina o predice nuevamente desde la tabla de gestión. Cuando el proyecto termine, finalízalo con los datos reales.

5

Mejora continua del modelo

Los proyectos finalizados alimentan el dataset de entrenamiento. Reentrena el modelo desde la interfaz para que las futuras predicciones sean más precisas.
