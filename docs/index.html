API Reference

Endpoints REST

API FastAPI documentada con validación automática vía Pydantic

GET / Health check — verifica que la API está en ejecución
POST /predict Predice riesgo general, sobrecosto y retraso para un proyecto
GET /opciones-formulario Obtiene opciones dinámicas del formulario (tipos, tecnologías, metodologías)
PUT /opciones-formulario Actualiza las opciones del formulario
POST /proyectos-ejecucion Registra un nuevo proyecto en ejecución
GET /proyectos-ejecucion Lista todos los proyectos en ejecución
GET /proyectos-ejecucion/{id} Obtiene datos de un proyecto específico
PUT /proyectos-ejecucion/{id} Actualiza los datos de un proyecto en ejecución
POST /proyectos-ejecucion/{id}/finalizar Finaliza un proyecto y lo mueve al dataset de entrenamiento
DEL /proyectos-ejecucion/{id} Elimina un proyecto en ejecución
POST /reentrenar-modelo Reentrena los modelos ML con los datos actualizados
POST /generar-reporte Genera y descarga un PDF profesional con la evaluación del riesgo
POST /enviar-reporte-mailhog Genera el PDF y lo envía por email vía MailHog
Machine Learning

Modelos de Predicción

Tres modelos XGBoost especializados entrenados con datos sintéticos realistas

🎯 Riesgo General

Clasificación multiclase (Alto, Medio, Bajo) del riesgo global del proyecto usando XGBClassifier con optimización de hiperparámetros vía GridSearchCV.

multi:softprob GridSearchCV 3 clases Upsampling

💰 Sobrecosto

Clasificación binaria que predice si el costo real superará al presupuesto estimado. Evalúa con ROC AUC y Log-loss.

binary:logistic ROC AUC Probabilístico

⏱️ Retraso

Clasificación binaria que predice si la duración real excederá al estimado del proyecto, usando el mismo set de features.

binary:logistic Log-loss Probabilístico

Features del Modelo

tipo_proyecto — Tipo de proyecto TI
metodologia — Scrum, Kanban, etc.
duracion_estimacion — Meses planificados
presupuesto_estimado — USD planificado
numero_recursos — Personas asignadas
tecnologias — Multi-hot encoded
complejidad — Baja, Media, Alta
experiencia_equipo — Años promedio
hitos_clave — Número de milestones

Instalación y Ejecución

Configura el entorno en minutos con estos pasos

1 Backend (API FastAPI)

# Clonar el repositorio
git clone https://github.com/MartinZapanaBerrospi/RiskPredictor-RPA.git

# Instalar dependencias Python
pip install -r requirements.txt

# (Opcional) Generar datos sintéticos
python data/generate_synthetic_data.py

# Entrenar los modelos
python models/train_xgboost.py

# Iniciar la API
uvicorn api.main:app --reload

📖 Documentación interactiva en http://localhost:8000/docs

2 Frontend (React + Vite)

# Entrar a la carpeta frontend
cd frontend

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

🌐 Interfaz disponible en http://localhost:5173

Requisitos: Python 3.8+ • Node.js 18+ • pip • npm
